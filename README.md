# Ular Tangga Matematika

Aplikasi game Ular Tangga interaktif untuk pembelajaran matematika siswa sekolah dasar.

## Cara Menjalankan

1. Buka file `index.html` di browser.
2. Klik **Lempar Dadu**.
3. Jawab soal matematika dengan benar agar bidak maju.
4. Pemain pertama yang mencapai kotak 100 menang.

## Fitur

- Papan Ular Tangga 10 x 10.
- Dua pemain bergiliran.
- Dadu interaktif.
- Soal matematika SD: penjumlahan, pengurangan, perkalian, pembagian, pecahan, satuan, dan soal cerita.
- Tangga untuk naik dan ular untuk turun.
- Tampilan responsif untuk laptop, tablet, dan ponsel.
